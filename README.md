# Personal Website

This is my personal website/portfolio showcasing my recent projects.

## Tech Stack

- React
- Bootstrap
- GitHub Pages

## Structure

- Navigation bar
- Body
- About Me
- Recent Projects
- Footer

## Setup

   ```bash
   npm install
   npm start
   ```

## Deployment

   ```bash
    npm run build
    npm run deploy
   ```
